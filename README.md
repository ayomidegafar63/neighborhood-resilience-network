# Neighborhood Resilience Network

## Overview

The Neighborhood Resilience Network is a comprehensive community preparedness and mutual aid platform designed to enhance neighborhood emergency response capabilities through blockchain technology. This decentralized system facilitates resource sharing, emergency coordination, and community recovery support during crisis situations.

## Description

Community preparedness and mutual aid network for emergencies that leverages blockchain technology to coordinate resources, responses, and recovery efforts in a transparent and efficient manner.

## Smart Contracts

### 1. Resource Inventory Contract
- **Purpose**: Inventory emergency supplies and skills across the neighborhood
- **Features**:
  - Community resource registration and tracking
  - Skills inventory and certification management
  - Resource availability and location mapping
  - Emergency supply distribution planning
  - Volunteer coordination and scheduling

### 2. Response Coordination Contract
- **Purpose**: Coordinate community response during emergency situations
- **Features**:
  - Emergency alert and notification system
  - Response team formation and assignment
  - Real-time status tracking and communication
  - Resource allocation and deployment
  - Incident management and documentation

### 3. Recovery Support Contract
- **Purpose**: Organize mutual aid and recovery support after emergencies
- **Features**:
  - Post-emergency needs assessment
  - Recovery resource allocation
  - Community support network coordination
  - Long-term recovery planning and tracking
  - Resilience improvement recommendations

## Technology Stack

- **Blockchain**: Stacks Blockchain
- **Smart Contracts**: Clarity
- **Development**: Clarinet
- **Testing**: Clarinet Test Framework

## Getting Started

### Prerequisites
- Clarinet installed
- Node.js and npm
- Git

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/okong7147/neighborhood-resilience-network.git
   cd neighborhood-resilience-network
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Check contracts:
   ```bash
   clarinet check
   ```

### Testing

Run the test suite:
```bash
clarinet test
```

## Contract Architecture

The system consists of three interconnected smart contracts that work together to provide comprehensive emergency preparedness and response:

1. **Resource Inventory** - Pre-emergency resource and skill mapping
2. **Response Coordination** - Active emergency management and coordination  
3. **Recovery Support** - Post-emergency mutual aid and rebuilding

## Usage

### Resource Registration
Community members register their available resources, skills, and availability for emergency response situations.

### Emergency Response
During emergencies, the system coordinates response efforts, allocates resources, and manages communication between community members.

### Recovery Planning
After emergencies, the platform facilitates mutual aid coordination, tracks recovery progress, and helps build long-term resilience.

## Key Features

- **Decentralized Coordination**: No single point of failure in emergency management
- **Transparent Resource Allocation**: Blockchain-based tracking ensures fair distribution
- **Community-Driven**: Empowers neighborhoods to manage their own emergency response
- **Interoperability**: Compatible with existing emergency management systems
- **Privacy-Conscious**: Protects sensitive information while enabling coordination
